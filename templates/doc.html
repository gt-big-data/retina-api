/article/
/article/id/<articleId>
Returns the full article in JSON format that matches a given ID.
Example: http://api.retinanews.net/article/id/55f02653a6b867b094a58f1e
/article/lasthours/<hours>
Returns all articles in the last <hours> hours.
Example: http://api.retinanews.net/article/lasthours/2
/article/sources
Get a list of all the sources available (cnn, bbc, reuters...)
Example: http://api.retinanews.net/article/sources
/article/source<source>/limit/<limit>
Returns the <limit> most recent articles from source <source>.
Example: http://api.retinanews.net/article/source/france24/limit/2
/article/keywords/<keywords>
<keywords> is a comma separated list of keywords (eg. kw1,kw2,kw3)
Returns a list of articles that match at least the keywords in the <keywords> list.
Example: http://api.retinanews.net/article/keywords/greece,refugee
/article/timeline/<keyword>/days/<days>
Builds a timeline (number of articles per timeslice) for the keyword <keyword> in the last <days> days.
Example: http://api.retinanews.net/article/timeline/obama/days/7
/article/graph/start/<start>/end/<end>
Returns a graph: list of vertices (articles) and edges (list of tuples of related articles).
<start> and <end> are UNIX timestamps
Example: http://api.retinanews.net/article/graph/start/1447525407/end/1447611807
/topic/
/topic/timeline/<topic>
Given a topic number, will return a timeline of counts of articles in that specfic topic per timeslice.
Example: http://api.retinanews.net/topic/timeline/2516
/topic/largest/days/<days>/limit/<limit>
Gives a list of the <limit> largest topics ids in the last <days> days.
Example: http://api.retinanews.net/topic/largest/days/3/limit/2
/topic/largestTimelines/days/<days>/limit/<limit>
Gives a list of timelines of the <limit> largest topics ids in the last <days> days.
Example: http://api.retinanews.net/topic/largest/days/3/limit/2
/topic/graph/<topic>
Gives a graph (set of nodes, set of edges) for a certain topic.
Example: http://api.retinanews.net/topic/graph/1
/tweet/
/tweet/start/<start>/end/<end>
[<start>, <end>]: UNIX timestamp range to fetch tweets from.
Example: http://api.retinanews.net/tweet/start/1447525407/end/1447611807
/tweet/timeline/<keyword>/days/<days>
Returns a timeline in each timeslice of the number of tweets that have the keyword <keyword> in the last <days>.
Example: http://api.retinanews.net/tweet/timeline/france/days/5
